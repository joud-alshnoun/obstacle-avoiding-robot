# obstacle-avoiding-robot
Arduino robot that avoids obstacles using ultrasonic sensor
# Components / Tools
- Arduino IDE
- C++ programming 
- Ultrasonic sensor
- DC motors
- # How I Built It
1. Connected ultrasonic sensor to Arduino pins 9 and 10.
2. Wired DC motors to motor driver pins.
3. Programmed the robot to move forward and detect obstacles.
4. If an obstacle is detected within 20 cm, the robot stops and turns right.
5. Tested and adjusted the motor timings for smooth navigation.
6. # Outcome
- The robot successfully avoids obstacles in a testing area.
- Smooth navigation and reliable obstacle detection.
